# my_simple_hyprpaw

Мой конфиг Hyprland + Waybar + Wofi + Wlogout + Hyprlock + Hypridle.

## Структура

```
hypr-dotfiles/
├── install.sh              # установщик: зависимости + симлинки + права
├── config/
│   ├── hypr/
│   │   ├── hyprland.lua    # основной конфиг Hyprland (Lua)
│   │   ├── hyprlock.conf   # экран блокировки
│   │   ├── hyprpaper.conf  # обои (универсальные, любое кол-во мониторов)
│   │   ├── hypridle.conf   # автовыключение экрана / блокировка / сон
│   │   ├── wallpapers/
│   │   │   └── paws-placeholder.png
│   │   └── scripts/
│   │       ├── keybinds.sh          # шпаргалка по хоткеям (wofi)
│   │       ├── logout.sh            # корректное завершение сессии
│   │       ├── lock-if-idle.sh      # блокировка с ожиданием конца медиа
│   │       └── dpms-off-if-idle.sh  # выключение экрана с ожиданием конца медиа
│   ├── waybar/
│   │   ├── config.jsonc
│   │   └── style.css
│   ├── wofi/
│   │   ├── config
│   │   └── style.css
│   └── wlogout/
│       └── layout
└── extras/
    └── JetBrainsMono.profile   # профиль шрифта для Konsole
```

## Установка на новую машину

```bash
git clone <URL-репозитория> ~/my_simple_hyprpaw
cd ~/my_simple_hyprpaw
./install.sh
```

Скрипт:
1. Ставит зависимости через `pacman` (Arch/EndeavourOS/Manjaro и т.п.). На других
   дистрибутивах спросит подтверждение и пропустит автоустановку — пакеты нужно
   поставить вручную (список пакетов — в начале `install.sh`).
2. Отдельно ставит `wlogout` из **AUR** (в официальных репозиториях Arch его нет) —
   через `yay` или `paru`, если один из них установлен. Если AUR-хелпера нет,
   выведет команды для установки вручную через `makepkg`.
3. Бэкапит существующие `~/.config/{hypr,waybar,wofi,wlogout}` в
   `~/.config-backup-<дата>` и создаёт симлинки на конфиги из репозитория.
4. Делает скрипты в `config/hypr/scripts/` исполняемыми.
5. Ставит профиль шрифта JetBrains Mono для Konsole (если Konsole установлена).

## Что унифицировано под разные машины

- **Мониторы** — `hyprland.lua` использует правило `output = ""`, `mode = "preferred"`,
  `position = "auto"`: Hyprland сам выбирает лучшее разрешение и максимальную частоту
  обновления для каждого подключённого монитора и расставляет их по порядку. Ничего
  не нужно менять руками при переносе на другой ПК. Если для конкретной машины нужно
  что-то зафиксировать вручную — пример закомментирован прямо в файле.
- **Обои** — `hyprpaper.conf` использует пустое поле `monitor`, то есть одна и та же
  картинка применяется ко всем мониторам сразу, независимо от их количества и имён
  (`HDMI-A-1`, `DP-3` и т.п. больше не захардкожены). По умолчанию используется
  нейтральная генерируемая текстура с лапками `wallpapers/paws-placeholder.png` —
  замените на своё фото, если хочется, путь менять не придётся.
- **Шрифт** — везде (waybar, wofi, hyprlock, Konsole) используется JetBrains Mono:
  моноширинный, с полной поддержкой кириллицы, есть в официальных репозиториях Arch
  (`ttf-jetbrains-mono`), поэтому подтягивается автоматически через `install.sh`.

## Ручные шаги после установки

- Перелогиниться в сессию Hyprland.
- В Konsole выбрать профиль `JetBrainsMono` как профиль по умолчанию
  (Настройки → Управление профилями).
- При необходимости — заменить `config/hypr/wallpapers/paws-placeholder.png`
  на своё изображение (путь в `hyprpaper.conf` менять не нужно, если оставить
  то же имя файла).
